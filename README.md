# Quick_Commerce_Data_Analysis
This project presents an end-to-end Exploratory Data Analysis (EDA) of a Quick Commerce dataset using Python. The analysis focuses on uncovering business insights related to sales performance, customer behavior, delivery efficiency, pricing, discounts, and operational metrics through data preprocessing, statistical analysis, and visualization.
- Performed end-to-end data cleaning, preprocessing, and exploratory data analysis (EDA) on quick commerce
order data using Pandas and NumPy.
- Conducted comparative analysis across companies, cities, customer segments, and product categories to identify
revenue drivers and operational trends.
- Engineered business KPIs including Revenue, Average Order Value (AOV), Customer Ratings, Delivery Time,
and Operational Efficiency to evaluate platform performance
